# 🛡️ Simulação de Ransomware Educacional

Este projeto é parte do desafio da formação em Cibersegurança da DIO e demonstra, de forma segura e educativa, o funcionamento básico de um ransomware com Python.

## 🚀 Funcionalidades

- Criptografia de arquivos com chave simétrica
- Descriptografia controlada com a chave gerada
- Separação em módulos reutilizáveis
- Totalmente seguro para fins didáticos

## 🛑 Aviso
Este projeto **não possui nenhum comportamento malicioso** e deve ser utilizado **apenas para fins educacionais**.

## 📂 Organização

- `criptografar.py`: criptografa arquivos
- `descriptografar.py`: descriptografa
